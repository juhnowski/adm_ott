Построение
==========
Перед запуском нужно прописать пути в файле src/main/java/ru/bellintegrator/Constants , а также при необходимости сменить порт
Эти данные влияют на первичную генерацию конфигурационного файла. Можно не прописывать, но после первого запуска сервера, будет сгенерирован
config.json который надо будет поправить вручную.
Затем перейти в корень проекта и выполнить: mvn package
Для построения компьютер должен иметь выход в интернет для скачивания зависимостей
После удачного построения выполнить в зависимости от ОС:
run.bat - для Windows
run.sh - для Linux
в браузере перейти по ссылке http://localhost:8080/edit

Конфигурация
============
Файл config.json содержит конфигурационные настройки:
"port":8080 - порт
"max_cnt":8 - количество позиций в дашборде
"max_sl_cnt":3 - количество афиш в каруселе
"img_path":"/home/user/adm_ott/src/main/resources/public/img" - путь к картинкам афиш
"zip_path":"/home/user/adm_ott/src/main/resources/public/zip" - путь к загруженным фотоархивам
"max_capture_len":25 - максимальная длина текста заголовка
"max_info_len":37 - максимальная длина текста информации

Правила заполнения
==================
Необходимо соблюдать следующие правила:
1) для первой позиции картинки
большая: 1150 × 575
маленькая: 600 × 300
2) для второй позиции
большая: 575 × 575
маленькая: 600 × 300
3) для всех остальных:
большая: 575 × 575
маленькая: 300 × 300
